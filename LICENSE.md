**🔴 Problem: Script executes but nothing happens in-game**

> ✅ **Fix:** The script may be targeting a different experience version. Verify the script is compatible with the current version of the popular title you are running it in. Try a basic print test first: `print("Arctic is working!")`.

**🔴 Problem: "bootstrapper.exe is not recognized" error in terminal**

> ✅ **Fix:** Run your terminal as Administrator. Right-click PowerShell → "Run as Administrator" → navigate to the Arctic folder and try again.

**🔴 Problem: Mobile web interface won't attach**

> ✅ **Fix:** Ensure you are using the correct browser (Chrome for Android, Safari for iOS). Clear your browser cache, reload the Arctic web page, and retry the pairing process. Disable any VPN during initial setup.

**🔴 Problem: Arctic crashes immediately after launch**

> ✅ **Fix:** Your .NET runtime may be outdated. Download and install the latest [.NET 8.0 Runtime](https://dotnet.microsoft.com/download) from Microsoft, then relaunch Arctic.

**🔴 Problem: UNC score is lower than expected**
